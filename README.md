# 1인미디어를 위한 NFT 기반 티켓 구매 어플

## 프로젝트 설명

- 아티스트/크리에이터 등이 전시나 북토크와 같은 현장(라이브) 미팅을 열 때, 티켓을 NFT로 발행하는 어플입니다.
- 거래 기한, 즉 티켓 판매 기한까지 NFT를 거래합니다.
- NFT를 소유한 사람들이 참여 가능합니다.
- 옥텟의 테스트넷과 API를 사용하여 구현하였습니다.
